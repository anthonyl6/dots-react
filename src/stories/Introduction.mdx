Dots UI is a React component library that provides a set of accessible, reusable, and composable components for building your React applications.

## Getting Started

To install Dots UI in your react project, you can run:

```bash
yarn add @dots-ui/react
# or
npm install @dots-ui/react
```

## Usage

```jsx
import { Button } from '@dots-ui/react';

const App = () => (
  <Button variant="primary">Hello World</Button>
);
```
This is the basic usage of Dots UI components. You can find more examples and details on [our website](https://dotsui.design/docs/introduction).
